# YouTube Clone 🎬

A full-stack YouTube-like video streaming web application built using **React.js** and **Tailwind CSS**.  
The project replicates core YouTube features with a clean, responsive, and scalable frontend architecture.

---

## 🚀 Features
- Video feed with dynamic content
- Search functionality
- Category-based filtering
- User subscriptions
- Responsive UI for all screen sizes
- Component-based architecture

---

## 🛠 Tech Stack
- **Frontend:** React.js, Vite
- **Styling:** Tailwind CSS
- **State Management:** React Hooks
- **Build Tool:** Vite
- **Linting:** ESLint
